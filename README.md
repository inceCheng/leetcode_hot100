# LeetCode Hot 100

本仓库用于记录和整理 LeetCode Hot 100 高频题的刷题过程、题解思路与代码实现。

目标不是单纯 AC，而是系统掌握常见算法题型，形成可复习、可迁移的解题方法。

## 仓库目标

- 完成 LeetCode Hot 100 全部题目
- 按题型整理核心解法
- 记录每道题的思路、复杂度与易错点
- 建立适合复习和面试准备的刷题笔记

## 题目分类

LeetCode Hot 100 主要覆盖以下算法与数据结构：

- 数组
- 哈希表
- 双指针
- 滑动窗口
- 字符串
- 链表
- 二叉树
- 图
- 回溯
- 二分查找
- 栈
- 堆
- 贪心
- 动态规划
